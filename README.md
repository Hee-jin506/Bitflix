# 👩‍💻 BITFLIX 🎬
비트캠프 자바 과정 1차 미니 해커톤에서 진행한 프로젝트입니다. (김하연, 최희진) 

## 주요 기능

- 멤버의 취향 장르에 따라 달라지는 화면 제공
- 추천 시스템
- 인기순, 장르별 영화 필터링
